Attention sous Windows !
  Bien configurer Git dans Windows pour faire fonctionner pub avec des dépendances git
  Configurer le chemin vers le répertoire package avec deux slashes au début et un à la fin (http://code.google.com/p/dart/issues/detail?id=4534)
    //C:/path/to/packages/
